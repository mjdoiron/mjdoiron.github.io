DmTower

DEMO (In-progress)

DmTower is hybrid web/mobile app for digitally augmenting in-person D&D games. The app is designed to quickly join/start games players on any device. Within the app, DMs will be able to send secret messages within the group or direct messages. Additionaly, DMs will be able to display images or video in a more cinematic way compared to traditional messaging apps.

App is created with Ionic/Angular and backend/hosting is via Firebase

AirMapper

AirMapper is the first iOS App built specifically for controlling electronic tabletop RPG maps. AirMapper is designed for the increasing number of gaming tables with built in TVs and projectors for displaying digital maps. AirMapper uses AirPlay or AV cable to control these maps directly instead of online gaming platforms like Roll20.

Current Features

  • Control map display wirelessly through Airplay
  • Control fog of war to reveal the map incrementally

Future Features

  • Stream via Google ChromeCast
  • Dropbox Integration
  • Save fog of war progress
  • Improving drawing performance on high-res maps

Built With

  • SwiftyDraw - A simple, lightweight drawing framework
  • Eureka - Elegant iOS form builder
  • SideMenu - Simple side menu control for iOS
  • Client: Personal
  • Date: April 2017
  • Service: iOS Development

Dungeon Dice Calculator

This is my first iOS app in the App Store using React-Native and Javascript. Inspired by the Netflix show "Stranger Things", I made a Dungeons and Dragons dice calculator. You are able to randomly simulate different sided dice (i.e., 4-sided, 6-sided, 20-sided, etc) and perform basic algebra using those random variables. I had a lot of fun learning React-Native. It is definitly going to be my go-to language for when I need a fairly simple app that I want to reach the most people (due to its cross-platform nature).

NMT Companion

IN PROGRESS

Inspired by a friend's neuro-music therapy company, I wanted to make an app the could assist a neuromusic therapist during therapy. The first tool is a simple metronome (often used in multiple therapies) that can change between different synthesized instruments. In additon, I am developing a second tool that quantifies a person's performance during melodic intonation therapy with the goal of being able to compare the amplitude and frequency of person's voice against a target audiotrack (similiar to vocals in the videogame Rock Band). The app utilizes the AudioKit framework, which is a great SWIFT framework for easily creating and manipulating Audio.

  • Client: Personal Project
  • Date: January 2017
  • Service: iOS Development
